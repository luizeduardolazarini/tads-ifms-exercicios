# 🚀 Lógica de Programação com JavaScript

Repositório dedicado aos exercícios de lógica de programação desenvolvidos durante o curso de **Tecnologia em Análise e Desenvolvimento de Sistemas (TADS)** no **IFMS**.

## 👨‍💻 Sobre o Projeto
Este projeto reúne uma série de algoritmos criados para consolidar conceitos fundamentais de programação, utilizando **JavaScript** e o ambiente de execução **Node.js**. Todos os exercícios foram testados via terminal no ambiente **Ubuntu (WSL2)**.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** JavaScript (ES6+)
- **Ambiente:** Node.js
- **Entrada de Dados:** Biblioteca `prompt-sync`
- **Versionamento:** Git & GitHub

## 📂 Lista de Exercícios
Abaixo estão os 10 desafios implementados, organizados por ordem de complexidade:

1.  **[01-guessing-game](./01-guessing-game)**: Jogo de adivinhação com números aleatórios.
2.  **[02-login-system](./02-login-system)**: Autenticação de usuário com limite de 3 tentativas.
3.  **[03-atm-system](./03-atm-system)**: Simulação de caixa eletrônico (Saldo, Depósito, Saque).
4.  **[04-user-registration](./04-user-registration)**: Cadastro com loop de confirmação de dados.
5.  **[05-sum-numbers](./05-sum-numbers)**: Acumulador de números com parada no zero.
6.  **[06-voting-system](./06-voting-system)**: Sistema de apuração de votos entre dois candidatos.
7.  **[07-password-validation](./07-password-validation)**: Validação de requisitos mínimos com `.length`.
8.  **[08-interactive-table](./08-interactive-table)**: Gerador automático de tabuada para um número específico.
9.  **[09-multi-tables](./09-multi-tables)**: Gerador contínuo de múltiplas tabuadas com loops aninhados.
10. **[10-inventory-control](./10-inventory-control)**: Controle de estoque com validação de regras de negócio (Desafio Final).
