# 🎮 Jogo de Adivinhação

Este é o primeiro exercício da lista de lógica de programação do curso de **Sistemas para Internet (TADS)**. O objetivo foi praticar estruturas de controle e repetição em JavaScript.

## 📝 Enunciado
Crie um algoritmo que gere um número de 1 a 10 e peça ao usuário para adivinhar. Use a estrutura `while` com `if/else`.

## 🛠️ Tecnologias e Conceitos Utilizados
- **JavaScript** (ES6+)
- **Node.js** (Ambiente de execução)
- **Biblioteca `prompt-sync`**: Para entrada de dados via terminal.
- **Estrutura de Repetição**: `while`
- **Condicionais**: `if / else`

## 🚀 Como Rodar o Exercício

Como este código foi desenvolvido para rodar no terminal (Ubuntu/WSL2), siga os passos abaixo:

1. Certifique-se de ter o Node.js instalado.
2. Instale as dependências:
   ```bash
   npm install prompt-sync
