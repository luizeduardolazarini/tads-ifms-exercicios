# 🔄 Gerador de Tabuadas Contínuo

Este é o exercício final da lista, integrando conceitos de aritmética, loops de repetição e controle de fluxo por entrada de usuário.

## 📝 Enunciado
Crie um algoritmo que:
1. Exiba a tabuada de um número (1 a 10).
2. Pergunte se o usuário deseja calcular outra tabuada.
3. Repita todo o processo até que a resposta seja negativa (N).

## 🚀 Conceitos Aplicados
- **Loops Aninhados (Nested Loops)**: Um ciclo de repetição dentro de outro para gerenciar tarefas e fluxos.
- **Estado de Continuação**: Uso de variáveis de texto para decidir a permanência no programa.
- **Modularização Logica**: Separação clara entre o cálculo e a interação de controle.
