# 🗳️ Sistema de Votação

Este exercício simula uma urna eletrônica simples, focando no uso de **incrementadores** para contabilizar dados em diferentes categorias.

## 📝 Enunciado
Crie um algoritmo com as opções:
- 1 – Candidato A
- 2 – Candidato B
- 0 – Encerrar votação
Ao final, exiba o total de votos de cada candidato.

## 🚀 Conceitos Aplicados
- **Incrementadores (`++`)**: Utilizados para somar uma unidade a cada novo voto.
- **Estrutura de Repetição `while`**: Mantém a votação aberta até que o valor de saída (0) seja digitado.
- **Saída de Dados Dinâmica**: Exibição dos resultados acumulados ao fim do processo.
