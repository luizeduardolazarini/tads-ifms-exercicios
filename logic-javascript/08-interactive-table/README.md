# 🔢 Tabuada Interativa

Este exercício demonstra como utilizar estruturas de repetição para automatizar cálculos aritméticos em sequência.

## 📝 Enunciado
Crie um algoritmo que:
1. Solicite um número ao usuário.
2. Gere a tabuada completa desse número (de 1 a 10).
3. Exiba os resultados de forma organizada no terminal.

## 🚀 Conceitos Aplicados
- **Iteradores**: Uso de uma variável de controle (`i`) para percorrer uma sequência numérica.
- **Operadores Aritméticos**: Realização de multiplicações dentro de cada ciclo do loop.
- **Template Strings**: Formatação da saída para exibir a conta completa (`A x B = C`).
