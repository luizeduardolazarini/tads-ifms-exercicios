# 🔐 Sistema de Login

Este exercício simula um sistema de autenticação básico, focado em praticar o controle de tentativas e operadores lógicos.

## 📝 Enunciado
Crie um algoritmo que solicite usuário e senha. Use a estrutura `while` com `if/else`.
- **Usuário correto:** admin
- **Senha correta:** 1234
- **Regra:** O sistema deve permitir até 3 tentativas antes de bloquear o acesso.

## 🛠️ Tecnologias e Conceitos
- **Variáveis de Controle:** Uso de contadores para limitar tentativas.
- **Template Strings:** Uso de interpoladores (ex: `` `Tentativas: ${tentativas}` ``) para mensagens dinâmicas.
- **Node.js**: Execução via terminal com `prompt-sync`.
- **Lógica de Booleanos**: Controle de estado para permissão de acesso.
