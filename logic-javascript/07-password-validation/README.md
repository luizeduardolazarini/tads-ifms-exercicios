# 🔐 Validação de Senha

Este exercício foca na manipulação de strings e validação de requisitos de entrada, simulando uma regra básica de segurança de sistemas.

## 📝 Enunciado
Crie um algoritmo que:
1. Solicite uma senha.
2. Valide se a senha possui no mínimo 6 caracteres.
3. Caso não atenda, solicite novamente até que o critério seja satisfeito.

## 🚀 Conceitos Aplicados
- **Propriedade `.length`**: Utilizada para contar a quantidade de caracteres em uma string.
- **Estrutura de Repetição `do-while`**: Ideal para garantir que a entrada ocorra antes da primeira verificação.
- **Lógica de Comparação**: Uso de operadores de menor que (`<`) para validação.
