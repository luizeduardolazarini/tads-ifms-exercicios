# ➕ Soma de Números

Este exercício exercita o conceito de **acumuladores**, onde uma variável é utilizada para guardar o estado progressivo de uma operação aritmética.

## 📝 Enunciado
Crie um algoritmo que:
1. Solicite números ao usuário.
2. Some todos os números digitados.
3. Encerre a execução quando o usuário digitar **0**.
4. Exiba o total da soma.

## 🚀 Conceitos Aplicados
- **Acumuladores**: Uso do operador `+=` para somar o valor atual ao valor armazenado.
- **Sentinela**: O valor `0` atua como uma condição de parada para o loop.
- **Validação com `isNaN`**: Garante que apenas entradas numéricas sejam somadas.
