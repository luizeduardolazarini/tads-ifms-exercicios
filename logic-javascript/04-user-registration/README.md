# 📝 Cadastro com Confirmação

Este exercício foca na validação de fluxo de dados, onde o sistema só encerra a execução após a confirmação explícita do usuário.

## 📝 Enunciado
Crie um algoritmo que:
1. Solicite **Nome** e **Idade**.
2. Pergunte: “Deseja confirmar os dados? (S/N)”.
3. Se a resposta for **N**, repetir o cadastro.

## 🚀 Conceitos Aplicados
- **Loops de Controle**: Repetição baseada em uma condição de texto.
- **Normalização de Strings**: Tratamento de entradas (maiúsculas/minúsculas).
- **Interação via Node.js**: Uso do terminal para entrada de dados.

node registration.js
