# Resolução da Lista P1 - Matemática Aplicada

Este repositório contém a resolução da Lista de Exercícios P1 da disciplina de **Matemática Aplicada**, do curso de TADS no **IFMS - Campus Nova Andradina**.

## 🚀 Sobre o Projeto

O objetivo é documentar o estudo de lógica de programação, operadores aritméticos e estruturas condicionais em Python.

## 📂 Exercícios Resolvidos

| Script | Descrição | Conceito Chave |
| :--- | :--- | :--- |
| `01_operadores.py` | Operadores básicos | Aritmética |
| `02_problemas.py` | Problemas aplicados | Lógica Matemática |
| `03_expressoes.py` | Expressões lógicas | Booleanos |
| `04_arquivos_por_pasta.py` | Divisão de arquivos | `//` e `%` |
| `05_combinacoes_bits.py` | Cálculo de bits | Potência `**` |
| `06_calculo_total_pixels.py` | Tamanho de imagem | Multiplicação |
| `07_verificacao_senha.py` | Login simples | Comparação `==` |
| `08_verificacao_autorizacao.py` | Permissões | Operador `or` |
| `09_analise_renda.py` | Crédito/Renda | Operador `and` |
| `10_espaco_disco.py` | Armazenamento | Conversão GB/MB |
| `11_download_tempo_resto.py` | Fluxo de rede | Tempo e Resto |
| `12_verificacao_tamanho_image.py` | Limite de arquivo | Comparação `>` |
| `13_verificacao_aprovacao.py` | Critérios de nota | Lógica Combinada |
| `14_verificacao_tempo_download.py` | Prazo de entrega | Comparação `<=` |
| `15_verificacao_compra_frete.py` | Regras de negócio | Precedência Lógica |
| `16_verificacao_aprovacao_nota.py` | Aprovação simples | `if/else` |
| `17_verificacao_velocidade.py` | Radar de trânsito | `if/elif/else` |
| `18_controle_saldo.py` | Sistema bancário | Atribuição `-=` |
| `19_identificacao_bits.py` | Arquitetura | Seleção de dados |
| `20_conversao_unidades.py` | MB para GB | Formatação de saída |

## 🛠️ Como Executar

No terminal (WSL2 ou Linux), utilize:
```bash
python3 nome_do_arquivo.py
```
